# About application

Gag scraper is application witch tracks hot posts for 24h and sends 10 most upvoted every day at 09:00 am (UTF +1) to mails from list of application subscribers.

Application consists of 2 parts: Node JS part witch job is to scrape and update posts in database, and part with php witch saves/deletes email from our database of subscribers.

This project is open source and everyone can help us improve it !
